# 黄巍Body！

大家平时有经常去体检吗？体检的时候我总感觉外科的医生比其他医生更热情……啊不是要说这个，体检可以预防疾病，大家1定要经常去体检哦！

体检之后，体检报告会告诉你哪些指标有异常，但是还有1个问题，它看不出哪些指标是在缓慢上涨的，比如说，我这几年是不是越来越胖了？

所以我写了1个parser，可以解析体检报告的pdf，这样1来就可以把历年的数据放在同1张markdown里，来看历年的身体指标变化啦！
